# Map Poster Marketing & Sales Strategy

## Product Overview

**Map Poster** is a web tool that generates personalized map posters from any location worldwide. Users enter a location, select a visual theme, and receive a custom map artwork in seconds.

## Target Audience

### Primary Personas

1. **Gift Givers**
   - Looking for unique, personalized presents
   - Occasions: weddings, anniversaries, housewarming, birthdays
   - Value: thoughtfulness, personalization, uniqueness

2. **Home Decorators**
   - Want meaningful wall art for their space
   - Value: aesthetics, personalization, uniqueness
   - Often millennials/Gen Z decorating first apartments

3. **Travelers & Nostalgics**
   - Want to commemorate places they've been
   - Value: memories, emotional connection, storytelling
   - Collect experiences, not things

## Copywriting Guidelines

### Tone of Voice

- **Warm** - not corporate or technical
- **Personal** - speak directly to the user ("tvůj", "tebe")
- **Emotional** - focus on feelings and memories
- **Confident** - no hedging or weak language

### Headlines Formula

1. **Emotional + Benefit:** "Proměň svá oblíbená místa v umění"
2. **Question + Pain Point:** "Které místo je pro tebe důležité?"
3. **Outcome + Speed:** "Hotovo za pár sekund"

### Microcopy Best Practices

| Element | Bad | Good |
|---------|-----|------|
| Location label | "Lokace" | "Které místo je pro tebe důležité?" |
| Location placeholder | "Zadej město" | "Praha, New York, Tokio..." |
| Theme label | "Téma" | "Vyber si styl" |
| CTA button | "Generovat" | "Vytvořit poster" |
| Loading | "Generuji..." | "Tvořím tvůj poster..." |
| Success | "Úspěch!" | "Hotovo! Tvůj poster je připraven." |

### Words to Avoid

- "Generovat" → use "Vytvořit" or "Tvořit"
- "AI-powered" → unnecessary tech jargon
- "Funkce" → focus on benefits, not features
- "Unikátní" → overused, prefer "osobní" or "na míru"

### Words to Use

- "Tvůj/Tebe" - personal ownership
- "Příběh" - emotional storytelling
- "Vzpomínka" - memory and nostalgia
- "Umění" - elevates the product
- "Na míru" - personalization

## Conversion Optimization

### Current Gaps

1. **No download/print CTA** after generation
2. **No email capture** for follow-up
3. **No urgency/scarcity** elements
4. **No pricing clarity** (free vs paid)

### Recommended CTAs After Generation

1. "Stáhnout poster" (primary)
2. "Objednat tisk" (secondary, if offering prints)
3. "Sdílet" (social sharing)
4. "Vytvořit další" (engagement loop)

## Competitive Positioning

### What Makes Map Poster Different

1. **Speed** - Instant generation vs. waiting for custom orders
2. **Simplicity** - No design skills needed
3. **Free to Try** - See the result before committing
4. **Czech Market** - Native Czech experience

### Against Competitors

| Competitor Type | Our Advantage |
|-----------------|---------------|
| Custom print shops | Faster, cheaper, instant preview |
| Generic map tools | Better themes, more artistic |
| DIY design tools | No skill required, instant result |
